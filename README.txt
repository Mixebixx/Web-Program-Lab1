## Описание проекта

Данное приложение реализовано на языке Python с использованием фреймворка FastAPI.
Приложение обрабатывает HTTP GET-запрос по маршруту `/info` и возвращает JSON-ответ с количеством дней до наступления нового года.

Пример ответа:

```json
{
    "days_before_new_year": 123
}
```

## Установка и запуск проекта

### 1. Локальный запуск

#### Установка зависимостей

```bash
python -m pip install fastapi uvicorn
```

#### Запуск сервера

```bash
python -m uvicorn main:app --reload --port 4200
```


### 2. Запуск через Docker

#### Сборка и запуск контейнера

```bash
docker compose up --build
```


## Проверка работы

После запуска откройте в браузере:

```
http://localhost:4200/info
```

Или выполните запрос через Postman.


## Структура проекта
